# notebookLM-java
# 📚 Estudos de Java com NotebookLM

> Projeto desenvolvido durante o **Bootcamp Bradesco**, utilizando o **NotebookLM** como ferramenta de apoio aos estudos da linguagem Java.

## 📖 Sobre o Projeto

Este repositório documenta meu processo de aprendizagem utilizando o NotebookLM para estudar a linguagem **Java**.

Durante o desenvolvimento do projeto, utilizei materiais de referência sobre Java para gerar resumos, esclarecer dúvidas, explorar conceitos da Programação Orientada a Objetos (POO) e praticar engenharia de prompts.

O objetivo não foi apenas aprender Java, mas também compreender como utilizar Inteligência Artificial de forma estratégica para potencializar os estudos.

---

## 🎯 Objetivos

* Aprender os fundamentos da linguagem Java.
* Compreender os conceitos de Programação Orientada a Objetos.
* Utilizar o NotebookLM como assistente de estudos.
* Praticar engenharia de prompts.
* Criar um material de revisão reutilizável.
